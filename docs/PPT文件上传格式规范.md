# PPT文件上传格式规范

## 支持文件格式与大小

支持上传文件格式为 **ppt、pptx、pdf** 的文件，要求文件大小 **＜20MB**。

| 序号 | 文件格式 | 文件大小 | 是否支持上传 | 备注 |
|------|---------|---------|------------|------|
| 1 | ppt | 15.2MB | ✅ | |
| 2 | ppt | 23MB | ❌ | |
| 3 | pptx | 15.2MB | ✅ | |
| 4 | pptx | 23MB | ❌ | |
| 5 | pdf | 15.2MB | ✅ | |
| 6 | pdf | 23MB | ❌ | |
| 7 | word | 15.2MB | ❌ | |
| 8 | excel | 15.2MB | ❌ | |
